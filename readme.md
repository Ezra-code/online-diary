 ## Description
 <p>This is a custom diary where a user is able to see all events that happened in their lives. each event is groupe according to the users feelings or mood that day</p>

## Features

As a user of the application,you will be able to :
* Select a feeling/mood and see a list of events associated with such
* see a list of feelings.
* upload a new event to the list
